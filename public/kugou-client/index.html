页面测试说明
给前端测试和第三方接入共用，先看流程，再看接口文档。
专辑链路 搜索类型选“专辑” -> 点击“查看歌曲” -> 选中歌曲播放 -> 检查封面、歌词、音频。
歌手链路 搜索类型选“歌手” -> 点击“查看歌曲” -> 选中歌曲播放 -> 检查 fallback 与无音源处理。
MV 链路 搜索类型选“MV” -> 先取 URL 再播放 MV -> 检查视频地址、封面与自动播放提示。
登录态链路 短信登录或二维码登录 -> 本地保存 token / userid / dfid -> 手动刷新 token -> 退出清空。
